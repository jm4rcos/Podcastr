# Podcastr


## 💻 Project
O # Podcastr é uma aplicação desenvolvida durante a Next Level Week 5 de React da RocketSeat. Nele podemos escutar nossos podcasts preferidos e acompanhar a postagem de novos. 

<br>
 
## ⛏ Techs
- [X] TypeScript
- [X] ReactJS
- [X] [Next.js](https://nextjs.org/docs)

### Dependencies
- [X] Yarn
- [X] Axios
- [X] JSON-Server
- [X] SASS
- [X] Date-Fsn

## 💻 Getting started

### Requirements

- You need to install both [Node.js](https://nodejs.org/en/download/) and [Yarn](https://yarnpkg.com/) to run this project.

**Clone the project and access the folder**

```bash
$ git clone https://github.com/felipeblobo/podcastr.git && cd podcastr
```

**Follow the steps below**

```bash
# Install the dependencies
$ yarn

# Run the backend fake server
$ yarn server

# Run the web server
$ yarn dev
```
